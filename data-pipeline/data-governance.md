# Data Governance - Compliant Private Transfers

**Created**: 2025-10-30 23:59:14  
**Version**: 1.0  
**Status**: Draft

---

## Policies
- Data minimization; no raw PII storage
- Access control: least privilege; audit trails for access
- Retention: rotate logs; expiry for disclosure bundles

## Compliance
- KYC via ICP II; OFAC/AML screening logs
- Travel rule metadata support where applicable

## Ownership & Stewardship
- Product owner approves schema changes
- Security team reviews data risk
