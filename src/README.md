# Source Overview

- adapters/: chain adapters (evm, sol, bnb, trx via unified interface)
- compliance/: KYC (ICP II), screening, disclosures
- prover/: zk proof generation/verification boundaries (no mixer semantics)
- ipfs/: IPFS/IPNS publish utilities
- pages/: UI routes (onboarding, dashboard, deposit, withdraw, relayers, auditor)
- lib/: shared utilities/types
